
# README #

Calc App

### This is a very simple calculator app implemented in Python and PyQt5 ###

## Technologies:
* [PyQt5](https://pypi.org/project/PyQt5/)

## Setup ###

## Installing Pipenv
```bash
pip install pipenv
```

## Install dependencies 
```bash
cd calc/calc_python_qt5
pipenv install
```

## Run app
```bash
python app.py
```
